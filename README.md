# ps-mockup-updates
PS样机切图工具更新仓库

## 更新日志 (Changelog)

### v2.2.0 - 稳定更新版 (最新)
- **界面优化**：大幅下调了整体窗口的启动尺寸与最小尺寸限制（从 820x560 降至 582x400），并在高分辨率下使用更紧凑的组件边距，完美适配 1080P （125%~150% 缩放）笔记本屏幕，不再溢出或留白。
- **强制更新保护**：新增了版本严重过期时的离线强制拦截体系。当检测到关键更新时，会在本地直接写入锁定标记。后续用户即使断网或试图被防火墙（如火绒）拦截也无法启动旧版本，必须允许联网下载新版后才可继续使用。

### v2.1.0
- 增强了极速模式的稳定性与错误状态下的自动恢复（断点续跑）。
- 移除了多模板功能，现在的逻辑更聚焦于单模板处理。
- 优化了退出体验：支持记忆关闭时的窗口大小和屏幕位置，下次启动自动恢复。
- 精简了极速模式的使用说明文字，布局更加清晰。
